# PolyU Career Coach

An AI-powered career coaching platform that helps users improve their resumes and advance their careers through personalized guidance.

## Features

**Resume Upload & Analysis**
- Upload PDF, Word, or text resumes
- AI-powered analysis with scoring across structure, content, and ATS compatibility
- Detailed feedback with specific improvement suggestions

**Interview Simulator**
- Practice interviews for specific positions
- AI-generated questions tailored to your role and resume
- Real-time feedback on answers with improvement suggestions
- Support for multiple interview rounds

**Pathway Advisory**
- Map academic profiles to potential career trajectories
- Identify skill gaps and development opportunities
- Personalized career path recommendations with match scoring
- Integration with resume analysis for enhanced suggestions

**Resource Navigator**
- Natural language queries for career information
- Salary data for Hong Kong market
- Job listings and opportunities
- Course recommendations and learning resources
- Company information and industry insights
- Skills requirements for various roles

## Getting Started

Backend: npm run dev:server
Frontend: npm run dev
